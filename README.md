# QuizeApp
An interactive and modern **Quiz Application built using React** that allows users to test their knowledge with multiple categories and difficulty levels.  
The app includes a countdown timer for each question and tracks the user's high score using `localStorage`.


 Select **category** & **difficulty** before starting the quiz
- **Timer per question** (auto move to next after timeout)
- **Progress indicator** with question numbers
- Score calculation at the end
- **High score saved locally**
- Detailed review of answers after the quiz
- Fully responsive & mobile friendly UI

  Tech Stack

| Technology | Purpose |
|-----------|---------|
| React.js | Component-based UI |
| React Router | Page routing |
| Context API | Global state management |
| CSS | Styling |
| LocalStorage | Saving high score |
